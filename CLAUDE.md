## Reglas de trabajo con git

- Este proyecto se sube a GitHub en https://github.com/julioborn/crc.git.
- A partir de ahora, Claude debe hacer commit y push automáticamente de los cambios sin pedir confirmación previa, salvo que el usuario indique lo contrario para una acción puntual.
- Usar mensajes de commit claros y concisos que expliquen el porqué del cambio.
- No usar `--force`, `--no-verify` ni otras banderas destructivas salvo pedido explícito del usuario.
